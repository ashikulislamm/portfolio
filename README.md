# Modern Developer Portfolio

This ismy personal portfolio website built with Next.js 14, showcasing full-stack development skills and projects. Features modern animations, responsive design, and seamless user experience.

## 🛠️ Technologies Used
- Next.js 14
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- Github Actions
